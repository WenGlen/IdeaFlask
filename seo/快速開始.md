## ⚙️ 自訂 SEO 資訊

如果您想修改 SEO meta 標籤：

1. 打開 `build-seo.js`
2. 找到 `metaTags` 部分（約第 120 行）
3. 修改內容：

```javascript
<meta name="description" content="這裡改成您的描述">
<meta name="keywords" content="這裡改成您的關鍵字">
<meta property="og:image" content="這裡改成您的分享圖片">
<meta property="og:url" content="https://您的網址.com">
```

4. 重新運行構建：`npm run build:seo`

## ❓ 疑難排解

### Q: 運行 `npm run build:seo` 時出現 "command not found"
**A:** 您需要先安裝 Node.js：https://nodejs.org/

### Q: 內容更新了但網站沒變化
**A:** 確保：
1. 已運行 `npm run build:seo`
2. 已用 `index-seo.html` 替換 `index.html`
3. 清除瀏覽器快取（Cmd/Ctrl + Shift + R）

### Q: 網站功能不正常
**A:** 檢查瀏覽器控制台是否有錯誤訊息，確認：
1. `js/load.js` 已正確更新
2. 所有 JSON 文件格式正確
3. 圖片路徑正確

### Q: 想恢復原版本
**A:** 
```bash
cp index-backup.html index.html
```
